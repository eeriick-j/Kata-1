# Proyecto Java - IntelliJ IDEA + Git

## Descripción
El proyecto es una práctica para aprender a trabajar con Java usando IntelliJ IDEA y Git, priorizando el uso de atajos de teclado y un flujo de desarrollo ordenado con commits frecuentes.

---

## Estructura del proyecto

```
src/main/java
    ├── Main.java
    └── Employee.java
```

---

## Objetivos de la práctica

- Crear un proyecto Java en IntelliJ IDEA
- Inicializar y usar Git desde el inicio del proyecto
- Crear clases Java dentro del proyecto
- Usar atajos de teclado para trabajar de forma más rápida
- Realizar commits frecuentes con mensajes claros

---

## Clases del proyecto

### Main.java
Clase principal donde se ejecuta el programa.

### Employee.java
Clase que representa un empleado con sus atributos básicos.

---

## Uso de Git

Se han realizado commits durante el desarrollo para registrar los cambios del proyecto.


## Nota
Este proyecto forma parte de una práctica de desarrollo para mejorar la productividad y el control de versiones.
